# Resistor Color

*Esercizio non ancora pronto...*

Le resistenze utilizzate in elettronica, hanno delle bande colorate che servono per identificare il loro valore in ohm.

Ogni colore ha un valore

- Nero: 0
- Marrone: 1
- Rosso: 2
- Arancione: 3
- Giallo: 4
- Verde: 5
- Blu: 6
- Viola: 7
- Grigio: 8
- Bianco: 9
